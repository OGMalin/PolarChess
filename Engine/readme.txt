
Parameters

** Pawn 

Change the value of a pawn in % of its default value.

** Knight

Change the value of a knight in % of its default value.


** Bishop

Change the value of a bishop in % of its default value.

** Rook

Change the value of a rook in % of its default value.

** Queen

Change the value of a queen in % of its default value.

** Contempt

Set to a high values will try to avoid draws. Negative valuse will make it prefere draws.
